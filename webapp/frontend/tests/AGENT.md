# Guía para agentes

- Propósito: pruebas end-to-end con Playwright para el frontend.
- Escenarios: `login-debug.spec.ts`, `api-keys.spec.ts`, `schedule-management.spec.ts`; guía adicional en `README.md`.
- Uso: ejecutar con `npx playwright test`; reportes generados en `playwright-report/` y `test-results/`.
- Requisitos: asegúrate de tener backend y datos semilla corriendo; credenciales configuradas en variables de entorno/test config.
