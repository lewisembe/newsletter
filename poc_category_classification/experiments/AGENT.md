# Guía para agentes

- Propósito: configuraciones YAML de experimentos de clasificación (umbrales, datasets, estrategias de ponderación).
- Archivos: `baseline.yml` y variantes `exp1`-`exp6` documentan ajustes probados; se cargan desde los scripts principales.
- Uso: duplica uno de los YAML y ajusta parámetros para nuevas corridas; conserva nombres descriptivos para rastrear resultados.
